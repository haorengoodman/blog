> 本篇文章目的在于普及MarkDown常用语法.

*斜体格式*

```
*斜体*   
_斜体_
```

**加粗格式**

```
**粗体**
__粗体__
```

***加粗斜体***

```
***加粗斜体***
___加粗斜体___
```

~~删除~~

```
~~删除线~~  
注：貌似删除线不大好用。目前markdown解析器使用的是：kramdown。
```

<del>删除线</del>  

```
<del>删除线</del>  
注：标签方式的删除线
```

**换行**  
在行末输入两个空格然后回车即可实现文本的强制换行

***标题***  
MarkDown 支持两种标题语法，类Setext和Atx形式  
类Setext形式采用底线的形式
一号标题
===
二号标题
---
类Atx形式采用的行首插入1到6各#，对应到标题1到标题6
# 一号标题
## 二号标题
### 三号标题
#### 四号标题
##### 五号标题
###### 六号标题

