**插入排序**  

![img](/img//img/Introduction_to_Algorithms/20151203.png)

```java
	public static int[] insertSort(int[] data) {
		for (int i=1;i<data.length;i++){
			int j = i;
			int temp = data[i];
			while(j > 0 && temp < data[j-1]){
				data[j] = data[j-1];
				j--;
			}
			data[j] = temp;
			printArr(data);
		}
		return data;
	}
```
